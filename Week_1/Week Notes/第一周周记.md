## 一周总结
----
* 对野指针和空指针所产生的bug有了更清晰的认知   
* 注册了**github**账号并上传了文件
  > 注：使用`git push`并尝试利用`git --set-upstream`创建远程分支
* 学习了一些git的知识，如
  1. 基本操作：
  * 初始化仓库 `git init`
  * 拷贝远程仓库 `git clone`
  * 添加文件到暂存区 `git add`
  * 提交文件到本地仓库 `git commit -m []`
  * 查看暂存区状态 `git status`
  * 查看历史记录 `git log`
  2. 分布管理：
  * 创建并转到分支（branch+checkout）`git check -b `
  * 删除分支 `git branch -d`
  * 合并分支 `git merge`
  3. 版本控制：
  * 版本回退 `git reset` `soft` or `hard`
